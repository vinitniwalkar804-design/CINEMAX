# CINEMAX---Movie-Theater-Seat-Booking-System

This project is developed to help customers to book the movie ticket from home.
Here i tried to develop a online movie ticket booking website which help the customer to book the ticket from home using an internet connection.
This website is designed with userfriendly, attractive and responsite User Interface which makes the ticket booking easy.
This project includes both user-side functionality and an authority/admin interface for managing bookings.

🚀 Features
  👤 User Interface
  🎥 View available movies on screen
  🪑 Select seats interactively
  🎟️ Book tickets easily
  📱 Responsive and user-friendly design
🛠️ Authority/Admin Interface
  📊 View all booked tickets
  🧾 Manage booking records
  🎫 Book tickets on behalf of customers
  🔄 Monitor seat availability


  🧩 System Overview

The system is divided into two main modules:

User Module
 > Allows customers to browse movies and book tickets
 > Provides seat selection and booking confirmation
   Authority Module
 > Acts as an admin panel
 > Manages bookings and maintains records
 > Can also perform bookings for customers

🛠️ Tech Stack

  > Frontend: HTML, tailwind CSS, JavaScript
  > Backend: Nodejs and Express
  > Database: Mongodb
